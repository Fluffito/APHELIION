# APHELION Store Listing Draft

## Extension name
APHELION

## Short description
Filter unwanted words and images with customizable blocking modes for a cleaner browsing experience.

## Full description
APHELION helps you clean up the web by filtering unwanted words, blurring or hiding flagged images, and giving you control over how blocked content appears.

### Core features
- Block custom words and phrases
- Blur, hide, or replace flagged images
- Choose custom censor glyphs
- Sync your preferences locally in the browser
- Unlock paid upgrades such as Unlimited Bonk and No-Ads Kitsune

### Paid features
- Unlimited word blocking
- Unlimited image blocking
- Bonk sound packs
- Custom sound uploads and profiles
- No-Ads Kitsune cosmetic support mode

### Privacy summary
APHELION stores settings and blacklist data locally in the browser. Stripe handles payments, and Supabase may be used for purchase recovery when enabled.
